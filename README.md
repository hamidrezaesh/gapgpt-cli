# 🤖 GapGPT CLI

A beautiful, feature-rich GapGPT chatbot CLI for terminal.

---

## ✨ Features

- 🎨 Beautiful terminal UI with colors and markdown
- ⌨️ Keyboard shortcuts (Ctrl+C, Ctrl+L, ↑/↓)
- 💾 Conversation memory and save/load to JSON
- 🔧 Global configuration for API key
- 📁 File reading with `@src(path/to/file)`
- 🌍 Cross-platform (Windows, Mac, Linux)

---

## 📦 Quick Start (3 commands)

```bash
# 1. Clone the repository
git clone https://github.com/hamidrezaesh/gapgpt-cli.git
cd gapgpt-cli
```

# 2. Install dependencies
```bash
# Install requirements
pip install -r requirements.txt
````

# 3. Run
```bash
# Run it
python gapgpt.py chat
```
